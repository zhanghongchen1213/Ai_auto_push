## Anthropic 推出 Claude Code 企业版

Anthropic 发布 Claude Code 企业版，面向大型开发团队提供代码审查、架构分析和自动化重构能力。企业版支持私有化部署，可与 GitHub Enterprise、GitLab 等平台深度集成。新增的团队协作功能允许多名开发者共享 AI 编程上下文，显著提升团队开发效率。

**来源：** [Anthropic Blog](https://www.anthropic.com/news/claude-opus-4-6)

## 微软发布 Phi-4 小模型：手机端运行的强大 AI

微软研究院发布 Phi-4 系列小模型，包含 1.5B 和 3.8B 两个版本。Phi-4 在同等参数规模下性能领先，3.8B 版本在多项基准测试中接近 7B 级别模型表现。模型针对移动端和边缘设备优化，支持在 iPhone 和 Android 设备上本地运行，推理延迟低于 100ms。

**来源：** [Azure Blog](https://azure.microsoft.com/en-us/blog/empowering-innovation-the-next-generation-of-the-phi-family/)

## Hugging Face 推出模型安全评估平台

Hugging Face 上线 AI 模型安全评估平台 SafetyBench，为开源模型提供标准化的安全性测试。平台覆盖偏见检测、有害内容生成、隐私泄露等 12 个安全维度，自动生成安全评分报告。目前已有超过 200 个热门模型完成评估，结果公开透明。

**来源：** [Hugging Face Blog](https://huggingface.co/blog/community-evals)

## NVIDIA 发布 NeMo Guardrails 2.0

NVIDIA 发布 NeMo Guardrails 2.0，为大模型应用提供更强大的安全护栏框架。新版本支持实时内容过滤、输出格式约束和多轮对话安全监控。框架与主流大模型 API 兼容，开发者可通过声明式配置快速部署安全策略，无需修改模型本身。

**来源：** [NVIDIA Blog](https://blogs.nvidia.com/blog/nemo-guardrails-nim-microservices/)
