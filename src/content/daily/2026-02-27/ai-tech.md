## Anthropic 发布 Claude Opus 4.6：百万级上下文窗口与 Agent Teams 协作能力

Anthropic 于2月5日正式发布旗舰模型 Claude Opus 4.6，距上一代 Opus 4.5 仅两个月，展现出极快的迭代节奏。该模型首次为 Opus 级别引入100万 token 上下文窗口（beta），支持最高128K输出 token，在 Terminal-Bench 2.0 上以65.4%的准确率领先同类模型。核心升级方向聚焦"自主性"与"任务持久性"——模型能更审慎地规划、更长时间维持代理任务、在大型代码库中更可靠地运行，并具备自我纠错的代码审查能力。同时推出的 Agent Teams 功能允许多个 AI 实例并行协作，标志着企业级 AI 工作流进入新阶段。

**来源：** [Anthropic 官方公告](https://www.anthropic.com/news/claude-opus-4-6)

## Google 发布 Gemini 3 Deep Think：面向科学研究的深度推理模式

Google DeepMind 于2月12日推出 Gemini 3 Deep Think 重大升级版本，这是一种专为复杂科学研究和工程应用设计的深度推理模式。与传统对话式 AI 追求响应速度不同，Deep Think 采用"系统2"思维架构，通过迭代式多轮推理同时探索多个假设路径来解决问题。该模式在数学、科学和竞赛编程基准测试中取得了创纪录的分数，达到奥林匹克数学金牌级别的推理能力。目前面向 Google AI Ultra 订阅用户开放，API 访问权限则提供给部分研究人员和企业用户，响应生成通常需要数分钟。

**来源：** [Google Gemini 发布说明](https://gemini.google/ca/release-notes/?hl=en-CA)

## OpenAI 推出 GPT-5.3-Codex-Spark：首个非 Nvidia 硬件部署的实时编程模型

OpenAI 于2月12日发布 GPT-5.3-Codex-Spark，这是其首个运行在 Cerebras 晶圆级引擎3（非 Nvidia 硬件）上的生产级模型，也是首个专为实时交互设计的 AI 编程模型。该模型在最优配置下可实现每秒超过1000个 token 的生成速度，使 AI 辅助编程的响应体验接近人类结对编程。在 Terminal-Bench 2.0 基准测试中达到77.3%的准确率，较前代 GPT-5.2-Codex 的64%有显著提升。Codex-Spark 支持实时代码编辑、逻辑重构和界面优化，目前面向 ChatGPT Pro 用户开放，可通过 Codex 应用、CLI 和 VS Code 扩展使用。

**来源：** [OpenAI 官方博客](https://openai.com/index/introducing-gpt-5-3-codex-spark/)

## Anthropic 完成300亿美元 G 轮融资，估值达3800亿美元创 AI 行业纪录

Anthropic 于2月13日宣布完成300亿美元 G 轮融资，由新加坡主权基金 GIC 和 Coatue 领投，D.E. Shaw Ventures、Dragoneer、Founders Fund、ICONIQ 及 MGX 共同参投，公司累计融资总额达691亿美元，估值飙升至3800亿美元。这是 AI 行业迄今为止规模最大的单轮融资，远超此前预期的100亿美元目标。Anthropic 同时披露预计2026年营收将达140亿美元，其中 Claude Code 产品在上线仅六个月后即实现10亿美元年化收入。此轮融资反映出资本市场对前沿 AI 基础设施的持续高度押注，也凸显了头部 AI 公司之间的竞争白热化态势。
