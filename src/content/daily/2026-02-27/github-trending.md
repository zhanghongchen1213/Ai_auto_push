## LadybirdBrowser/ladybird

Ladybird 是一款完全独立的开源网页浏览器，不依赖任何现有浏览器引擎，从零开始构建。项目主要使用 C++ 和 Rust 编写，目前拥有约 58.6k Star。近日 Ladybird 宣布借助 AI 辅助将 JavaScript 引擎从 C++ 迁移至 Rust，仅用两周即完成移植且零回归。首个 Alpha 版本计划于 2026 年发布，支持 Linux 和 macOS。该项目获得 Cloudflare 赞助，是当前最受关注的独立浏览器项目，对打破 Chromium 垄断具有重要意义。

**来源：** [GitHub](https://github.com/LadybirdBrowser/ladybird)

## abhigyanpatwari/GitNexus

GitNexus 是一个零服务器的代码智能引擎，完全在浏览器端通过 WebAssembly 运行。用户只需拖入 GitHub 仓库链接或 ZIP 文件，即可自动生成交互式代码知识图谱，映射出每个依赖关系、调用链、代码集群和执行流程。内置 Graph RAG Agent 支持基于图谱的智能问答。项目于 2 月 22 日登上 GitHub Trending，短短几天获得超过 1200 Star。它解决了传统 RAG 在代码理解上的不足，被称为"比 DeepWiki 更深入"的代码分析工具，值得 AI 编程领域开发者关注。

**来源：** [GitHub](https://github.com/abhigyanpatwari/GitNexus)
