## trendshift.io — GitHub Trending 替代分析平台

Trendshift 是一个基于每日参与度指标的开源项目发现平台，提供比 GitHub 官方 Trending 更稳定的评分算法。截至 2026-02-26，平台持续追踪数千个活跃仓库，按日更新热度排名。对于开发者而言，Trendshift 能有效过滤短期刷榜项目，聚焦真正有持续社区活跃度的仓库。平台支持按语言、领域筛选，并提供 star 增长趋势图，是发现优质开源项目的高效工具。

**来源：** [Trendshift](https://trendshift.io/)

## GitHub CLI trending browser — 命令行浏览 GitHub 热门项目

一个用 JavaScript 编写的命令行工具，让开发者无需打开浏览器即可直接在终端浏览 GitHub Trending 项目列表。该项目于 2026 年 2 月 16 日发布，已获得 580 个 Star，增长迅速。工具支持按语言过滤、查看项目描述和 Star 数，适合习惯在终端工作的开发者日常使用。项目托管于 GitHub Topics 的 trending-repositories 分类下，是近期开发者工具类开源项目中的热门新秀。
