## OpenClaw（原 Clawdbot）

OpenClaw 是 2026 年初最炸裂的开源项目，60 天内从 9,000 星飙升至 188,000 星。它是一个自托管的 AI 代理网关，将 GPT-5、Claude、DeepSeek 等大模型直接连接到本地操作系统和消息应用（WhatsApp、Telegram、Discord）。基于 Node.js 长驻进程架构，支持本地 Markdown 文件存储记忆，真正实现了"AI 助手住在你电脑里"的愿景。Wikipedia 已为其建立独立词条，生态衍生项目 PicoClaw、NanoClaw 同步走红。

**来源：** [BuildMVPFast](https://www.buildmvpfast.com/blog/best-open-source-ai-projects-github-2026)

## Ollama

Ollama 是本地运行大语言模型的首选工具，GitHub 星标已突破 162,000。支持一行命令拉取并运行 LLaMA 3.3、Qwen 2.5、DeepSeek-R1 等主流开源模型，无需复杂配置即可在消费级硬件上跑推理。与 OpenClaw 深度集成后，成为本地 AI Agent 的标准推理后端，开发者可以零成本搭建完全私有的智能助手系统，隐私和成本优势显著。

**来源：** [BuildMVPFast](https://www.buildmvpfast.com/blog/best-open-source-ai-projects-github-2026)

## Dify

Dify 是一个开源的 LLM 应用开发平台，星标突破 130,000。提供可视化的 Prompt 编排、RAG 管道、Agent 工作流和模型管理能力，让非技术人员也能快速搭建 AI 应用。支持对接 OpenAI、Anthropic、本地 Ollama 等多种模型后端，内置知识库管理和 API 发布功能。2026 年持续迭代，已成为企业级 AI 应用开发的事实标准平台之一，社区活跃度极高。

**来源：** [BuildMVPFast](https://www.buildmvpfast.com/blog/best-open-source-ai-projects-github-2026)

## obra/superpowers

Superpowers 是由 Jesse Vincent 创建的 Agentic Skills 框架，为 Claude Code 等 AI 编程助手提供可复用的技能插件体系。1 月中旬单日新增 2,053 星，总星标超过 25,000。它定义了一套标准化的 Skills 编写规范，让开发者可以为 AI 代理创建、分享和组合各种专业能力。2 月底 Skills 生态正式从个人实验阶段进入平台级支持，与 HuggingFace Skills 形成双轨并行格局。

**来源：** [GitHub](https://github.com/obra/superpowers)

## huggingface/skills

HuggingFace 官方推出的 Skills 仓库，一周内新增 3,381 星，标志着 AI Agent 技能生态从实验走向主流。该仓库提供标准化的指令模板、脚本和工具集，让开发者可以为各类 AI 代理编写可移植的技能包。与 obra/superpowers 和 Agent-Skills 共同构成了 2026 年 Skills 生态的三驾马车，推动 AI 编程助手从"单一对话"进化为"多技能协作"的新范式。

**来源：** [ShareUHack Weekly](https://www.shareuhack.com/en/posts/github-trending-weekly-2026-02-25)

## Google A2A（Agent-to-Agent）协议

Google 开源的 A2A 智能体互联协议，发布不到一周 Star 数即将破万。继 Anthropic 的 MCP 协议之后，又一科技巨头推出的 LLM 应用互联标准。A2A 定义了 AI Agent 之间的通信规范，让不同厂商、不同框架的智能体可以互相发现、协商和协作。这是 AI 基础设施层的重要里程碑，预示着多智能体协作将成为 2026 年的核心趋势。

**来源：** [博客园 GitHub热点速览](https://www.cnblogs.com/xueweihan/p/18825970)

## SesameAILabs/csm

Sesame AI Labs 发布的对话式语音生成模型（Conversational Speech Model），基于 LLaMA 架构并增加 Mimi 音频解码器，可从文本和音频输入生成高质量 RVQ 语音编码。采用双 Transformer 自回归架构，在 HuggingFace 发布了 1B 参数版本。开源后在 Hacker News 和开发社区引发热议，衍生出 csm-streaming（实时音频生成）等增强项目，被视为跨越语音交互"恐怖谷"的重要突破。

**来源：** [GitHub](https://github.com/SesameAILabs/csm)

## SenWeaver IDE

SenWeaver IDE 是一个开源多智能体协同 AI 编程平台，内置自主在线强化学习（APO）引擎。系统从每次用户交互中实时学习，可自建知识库进行训练，自动优化 Prompt 策略，越用越智能。支持主 Agent + 子 Agent 并行协作、40+ 内置工具（文件/终端/搜索/浏览器/视觉）、AI 代码安全检测、WebRTC P2P 远程协作，兼容 GPT/Claude/DeepSeek/Ollama 等多模型。MIT 协议完全开源。

**来源：** [GitTrends Weekly](https://gitstars.substack.com/p/gittrends-february-16-2026-openclaw)

## Moondream

Moondream 是一个轻量级视觉 AI 模型，被 GitHub 评为年度最具影响力开源项目之一。拥有超过 600 万次下载和数千 GitHub 星标，在医疗、机器人等行业广泛应用。它将最先进的视觉 AI 能力压缩到极小的模型体积中，开发者只需一行代码即可集成图像理解能力。作为边缘端视觉 AI 的代表项目，Moondream 证明了小模型在特定场景下同样可以达到生产级质量。

**来源：** [GitHub Blog](https://github.blog/open-source/maintainers/this-years-most-influential-open-source-projects/)

## browser-use/web-ui

Browser-Use 是一个让 AI Agent 直接在浏览器中运行的开源项目，提供 Web UI 界面来控制 AI 代理执行网页自动化任务。支持网页导航、表单填写、数据抓取、截图等操作，是 2026 年 AI Agent 工具链中"眼睛和手"的关键组件。随着 AI Agent 从纯文本交互向多模态操作演进，浏览器自动化成为刚需，browser-use 凭借简洁的架构和活跃的社区迅速走红 GitHub Trending。

**来源：** [GitHub](https://github.com/browser-use/web-ui)
