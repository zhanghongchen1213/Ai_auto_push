## GitHub 发布 REST API 2026-03-10 版本，新增多项开发者功能

GitHub 正式发布 REST API 2026-03-10 版本，这是采用日历化版本号后的首次重大更新。新版本新增了细粒度的仓库权限管理接口、AI 相关 API、增强的 WebHook 事件系统等功能，并提供了长达 24 个月的支持周期。GitHub 表示将每三个月发布一个新的 API 版本，同时为旧版本提供充足的升级过渡期。开发者需要升级应用以适配新的 API 规范，旧版本 API 计划于 2028 年 3 月正式停用。

**来源：** [GitHub Blog](https://github.blog/changelog/2026-03-12-rest-api-version-2026-03-10-is-now-available/)

## GitHub Copilot 学生套餐调整，移除 GPT-5.4 和 Claude 模型直接访问权限

GitHub 宣布调整 Copilot 免费学生套餐，自 3 月 12 日起，学生用户将无法直接选择 GPT-5.4、Claude Opus、Claude Sonnet 等高级模型，这些模型将被限制为仅自动模式可用。官方表示这一调整是为了优化资源分配，降低运营成本，学生用户仍然可以通过自动模式体验所有模型的能力，只是无法手动指定模型版本。这一调整引发了学生开发者群体的广泛讨论，部分用户认为会影响学习和开发效率。
