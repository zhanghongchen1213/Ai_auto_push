# Product Brief: Ai_auto_push

## Executive Summary

Ai_auto_push 是一个面向个人使用的全自动化多领域智能资讯聚合平台。通过 Claude Code Skills 技能包驱动 openclaw + 大模型的组合，实现零人工干预的资讯抓取、AI 整理与自动发布。

与现有单一领域资讯站（如 AI News）不同，Ai_auto_push 支持跨领域内容聚合（AI、跨境电商、产品创业、GitHub 热门项目等），通过开放的 Skills 架构实现内容源的灵活配置与扩展。系统以 Git 仓库作为中间层，openclaw 生成统一格式的 Markdown 文档后自动推送至仓库，触发静态网站自动构建与部署。

核心资产是 Claude Code Skills 技能包，它决定了抓取工具配置、领域选择、内容整理规范和发布推送策略，形成完整的信息自动化闭环。

## Core Vision

### Problem Statement

当前获取多领域资讯的方式极度碎片化。用户需要在视频平台、公众号、新闻网站等多个渠道之间来回切换，耗费大量时间和精力。即使是优秀的资讯聚合站（如 AI News），也仅聚焦单一领域，无法满足同时关注 AI、跨境电商、创业、开源项目等多元需求的用户。

### Problem Impact

- 信息获取成本高：每天需要浏览多个平台才能获得完整的信息视野
- 信息过载与遗漏并存：要么被海量推送淹没，要么错过关键动态
- 无法个性化定制：现有平台的内容源和领域固定，用户缺乏控制权
- 效率瓶颈：在 AI Agent 时代，信息索取能力直接决定了 AI 使用的上限


### Why Existing Solutions Fall Short

- **AI News 等垂直聚合站**：内容质量高但领域单一，无法跨域聚合，无法个性化定制
- **公众号/视频平台**：被动推送，信息碎片化，无法结构化浏览
- **RSS 阅读器**：需要手动配置，缺乏 AI 整理和摘要能力
- **搜索引擎**：主动搜索模式，无法实现每日自动推送和趋势追踪


### Proposed Solution

构建一个基于 Claude Code Skills + openclaw + 大模型的全自动资讯平台：

- **Skills 技能包（核心引擎）**：Claude Code Skills 驱动全链路，决定抓取配置、领域选择、整理规范和发布策略
- **自动抓取**：openclaw 按 Skills 配置定时从多领域信息源抓取资讯
- **AI 整理**：大模型将原始信息整理为统一格式的 Markdown 文档
- **Git 推送**：md 文档自动推送至 Git 仓库（Ai_auto_push）
- **自动部署**：静态网站监听仓库变化，自动构建并发布
- **历史回溯**：用户可浏览任意日期的历史资讯


### Key Differentiators

1. **Skills 驱动架构**：Claude Code Skills 技能包作为核心资产，反向指导 openclaw 的抓取策略，实现声明式信息管道配置
2. **多领域个性化聚合**：不局限于单一领域，支持 AI、跨境电商、创业、GitHub 等多元内容源
3. **全链路自动化**：从抓取到整理到发布，零人工干预
4. **精选而非轰炸**：每日精选摘要，极简晨报体验，30秒了解当天关键动态
5. **低成本高可靠**：md 文件即数据库，Git 仓库即中间层，静态站点零运维成本
6. **时代红利**：AI Agent 时代的个人信息基础设施，信息索取能力即竞争力

## Success Metrics

- **替代传统信息渠道**：完全替代公众号、视频号等碎片化资讯获取方式，成为每日唯一的信息入口

- **可扩展性**：通过 Skills 机制随时添加新领域，无需修改代码或重新部署

## MVP Scope

### Core Features

1. **每日资讯展示**
   - 按日期分组，每天一个资讯页面
   - 每条资讯包含：标题、摘要（150-200字）、原文链接
   - 参考 ai-bot.cn/daily-ai-news 的时间轴纵向排列设计

2. **多领域分区展示**
   - 四大领域分区：AI 技术、跨境电商、产品创业、GitHub 热门项目
   - 每个领域独立区块，视觉上清晰区分
   - 领域由 Skills 技能包配置驱动

3. **历史资讯浏览**
   - 时间线导航，支持按日期快速跳转
   - 浏览任意历史日期的资讯内容

4. **搜索功能**
   - 按关键词搜索历史资讯
   - 支持跨日期、跨领域的全文检索

5. **移动端适配**
   - 响应式设计，手机浏览体验良好

6. **Skills 技能包（Claude Code Skills）**
   - 抓取层：openclaw 抓取配置
   - 领域层：AI、跨境电商、产品创业、GitHub 四个领域
   - 整理层：统一 md 格式规范
   - 发布层：Git 推送触发自动部署

### Out of Scope for MVP

- 用户注册/登录系统
- 资讯收藏/书签功能
- 邮件/微信推送通知
- 多用户个性化订阅
- 评论/互动功能
- 多语言支持

### MVP Success Criteria

- 系统连续 7 天稳定运行，每日 8:00 后自动更新资讯
- 四个领域每天均有资讯输出，无遗漏
- 资讯日期 100% 准确，原文链接 100% 可用
- 网站在 PC 和手机端均可正常浏览
- 搜索功能可正确检索历史资讯

### Future Vision

- **Phase 2**：用户系统 + 收藏书签 + 推送通知
- **Phase 3**：多用户个性化订阅 + 更多领域扩展
- **长期愿景**：从个人工具演进为开放的多领域资讯聚合平台
