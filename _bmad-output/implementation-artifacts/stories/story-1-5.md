# Story 1-5: 领域筛选与色彩编码

## Story ID
1-5

## Epic
Epic 1: 项目基础与每日资讯展示

## 标题
领域筛选与色彩编码

## 描述
作为用户，
我想要通过点击领域 pill 标签筛选特定领域的资讯，
以便快速聚焦到我当前最关心的领域动态。

## 优先级
P1

## 复杂度
低

## 状态
in-progress

## 依赖
- Story 1-4（每日资讯首页与新闻卡片组件）

---

## 验收标准 (Acceptance Criteria)

### AC-1: 领域 pill 标签筛选交互
**Given** 首页已展示全部领域资讯（Story 1.4）
**When** 用户点击某个领域 pill 标签（如"AI技术"）
**Then** 资讯列表仅显示该领域的资讯条目
**And** 被点击的 pill 标签切换为激活态（背景 #1677FF，文字白色）

### AC-2: "全部"标签恢复全部显示
**Given** 用户已筛选某个领域
**When** 用户点击"全部"标签
**Then** 恢复显示所有领域资讯

### AC-3: 客户端即时响应与过渡动画
**Given** 用户点击 pill 标签
**When** 筛选切换发生
**Then** 无需页面刷新，客户端即时响应
**And** 使用 CSS transition 实现平滑的显示/隐藏效果

### AC-4: URL 参数同步
**Given** 用户点击某个领域 pill 标签
**When** 筛选状态改变
**Then** URL 参数同步更新（如 ?domain=ai-tech）
**And** 直接访问带参数的 URL 可恢复对应筛选状态

### AC-5: 色彩编码一致性
**Given** 资讯卡片已渲染
**When** 用户查看卡片上的领域 pill 标签
**Then** 卡片内领域 pill 标签颜色与 Header 导航中的领域色彩一致

### AC-6: 键盘无障碍导航
**Given** 键盘用户使用 Tab 进入导航区
**When** 用户按左右箭头键和 Enter/Space
**Then** 焦点在 pill 标签间切换，按键可激活筛选
